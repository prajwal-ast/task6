# 🔐 Task 6 – Password Strength Evaluation

## 📌 Overview
Evaluated passwords of varying complexity using PasswordMeter to understand how password length, character diversity, and predictability affect overall security.

## 🛠️ Tool Used
- PasswordMeter
- Web Browser

## 📊 Evaluation Summary
- Tested passwords with different complexity levels.
- Compared password strength and estimated crack times.
- Identified weaknesses such as common passwords, keyboard patterns, and predictable years.
- Learned password security best practices.

## 🎯 Outcome
Successfully analyzed password strength and understood the importance of creating long, unique, and complex passwords to defend against common password attacks.
